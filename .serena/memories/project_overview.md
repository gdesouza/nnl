NNLang (nnl) is a single-Rust-crate project that provides the `nnc` compiler for converting `.nnl` neural network definitions and supported ONNX imports into standalone native artifacts. The main binary is `nnc` (`src/main.rs`), with `driver::run` in `src/driver.rs` as the command entrypoint. The core compiler pipeline flows through `src/syntax/`, `src/sema/`, `src/ir/`, `src/weights/`, and `src/codegen/`. The repo also contains mdBook docs in `docs/`, static website assets in `website/`, tests in `tests/`, and example models in `examples/`. The project is systems-oriented: generated inference code uses static allocation, embeds weights, and targets standalone executables or C ABI artifacts.
