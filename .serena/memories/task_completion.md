Before finishing code changes, try to match CI expectations from `AGENTS.md`: run `cargo fmt --check`, `cargo clippy --all-targets -- -D warnings`, and `cargo test` when relevant to the touched code. For docs/website/example work, at least validate the affected files directly (for example, parse HTML, check Python script syntax, run Makefile/example commands when feasible). Be aware that `nnc compile` shells out to the host C toolchain, and `nnc test` depends on a working C compiler.
