Code style is pragmatic and minimal. Rust is the primary implementation language; example and website content live in plain files without a frontend build step. Repo guidance emphasizes small correct changes, avoiding unrelated edits, and matching existing patterns. For website work, match `website/index.html` and `website/styles.css` rather than introducing frameworks. For repo exploration, start with `README.md`, `docs/src/cli.md`, `docs/src/codegen.md`, and `tests/compile.rs`. Avoid treating `target/` or `docs/book/` as source.
